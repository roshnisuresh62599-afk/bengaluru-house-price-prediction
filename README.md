# Bangalore House Price Prediction

## Project Overview
Machine learning project to predict house prices in Bangalore using housing dataset.

## Dataset
- 13,000+ rows
- Features include:
  - Location
  - Total Sqft
  - Bath
  - Balcony
  - BHK
  - Price

## Steps Performed
- Data Cleaning
- Missing Value Treatment
- Feature Engineering
- Outlier Removal
- Exploratory Data Analysis
- Correlation Heatmap
- Linear Regression Model

## Model Performance
- R² Score: 0.65
- MAE: 31.4 Lakhs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## Output
Predicted Bangalore house prices using trained model.
